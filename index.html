Three Column Rows

So this page's display is responsive. At media size 993 and up (most computer screens), it has three columns. The CSS looks like this: 
/* Container Element */
.columns {
  display: grid;
  grid-template-columns: 33% 33% 33%;
  justify-content: center;
  row-gap: 15px;
}

Smaller Displays

In other media sizes though, the display changes. In the 'table' screen range, it has two columns, and on mobile devices it has one.

We do this by adding the following to the bottom of the stylesheet: 

/* Media Queries */
@media (max-width: 992px) {
  .columns {
    grid-template-columns: 50% 50%;
  }
}

@media (max-width: 768px) {
  .columns {
    grid-template-columns: 100%;
  }
}

Box Transition Glow

How about that cool on-hover box glow? Transitions and box-shadow my friend... 
/* Assign a Transition Standard to the Elements in Question */
.card, .card-header {
    transition: all 0.2s ease-in-out;
}

/* Set Up the On-Hover Effect */
.card:hover,
.card:hover .card-header {
  box-shadow: 
    inset 0px 0px 8px #pink,
    0 0 15px #pink;
}

Tooltips pt.1

How about those handy blue tooltips?

This one's a little lengthy, but not too tricky once you understand it 

/* Set up base class */
.hover-for-info {
  position: relative;
  cursor: pointer;
  color: #blue;
}

/* Add some visual pop */
.hover-for-info:hover {
  color: #yellow;
}

/* Here's the good stuff */
.hover-for-info:hover::after {
  position: absolute;
  background-color: #blue;
  left: 0px;
  bottom: 24px;
  min-width: 200px;
  border: 2px white solid;
  border-radius: 5px;
  padding: 12px;
  font-size: 80%;
  z-index: 1;
}

Tooltips pt.2

Okay, so the setup is done. Now how do we implement? 

/* In HTML, we write something like this */
<span class="hover-for-info"
    id="final-product"
>
final product
</span>
/* Then in CSS we add content via the Id */
#final-product:hover::after {
    content: 'Hooray!';
}

/* And we get our final product! */
